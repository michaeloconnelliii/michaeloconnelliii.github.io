Brevet Time Calculator

  • Github

The Randonneurs USA (RUSA) time calculator is a tool used for calculating control times for a brevet. A brevet is a timed, long distance road cycling event.

The calculator consists of three parts: A main web server, an API and a database. All of which are deployed using Docker and Docker Compose. The webpages are built using HTML, CSS, and Javascript. The main web server and API is built using Python and uses Flask as their framework. The API retrieves and serves open and close times from the database (MongoDB). A sample program is also included to show a demo of displaying the open and close times retrieved using the API.

The biggest challenge I faced during this project was getting all the pieces/services to cohesively work together. Through this project, I learned how to make a small, full-stack web application.
